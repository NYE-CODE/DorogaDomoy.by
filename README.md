# 🐾 Платформа поиска пропавших домашних животных

Веб-платформа для быстрого поиска пропавших домашних животных с интерактивной картой.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр production сборки

```bash
npm run preview
```

## 📦 Технологии

- **React 18** + **TypeScript**
- **Tailwind CSS v4** - стилизация
- **Leaflet** - интерактивные карты
- **Lucide React** - иконки
- **Sonner** - уведомления
- **Vite** - сборщик

## 📂 Структура проекта

```
src/
├── components/          # React компоненты
│   ├── auth/           # Аутентификация
│   ├── layout/         # Компоненты layout
│   ├── ui/             # UI библиотека
│   └── ...             # Основные компоненты
├── context/            # React Context
├── data/               # Mock данные
├── types/              # TypeScript типы
├── utils/              # Утилиты
└── styles/             # Глобальные стили
```
